A BepInEx mod that prevents Console from sending telemetry data and accessing the internet.
This currently blocks a majority of endpoints on iidk.online, but not the friend system.

# Info
-----------------------------
Console Sends this Data once you join a room:

1. Room Name
2. Nickname
3. Player ID
4. Room Visiblity
5. Room's PlayerCount
6. Gamemode
----------------------------
This blocks that! And makes it so it blocks the http sends.
